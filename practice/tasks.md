# Завдання 1. Визначення безпечного пароля

Оберіть пароль, який є найбезпечнішим:

1. 12345678
2. qwerty
3. password
4. G7#kL9!pQ2

## Питання:
- Який пароль є найбезпечнішим?
- Чому інші паролі є слабкими?

# Завдання 2. Аналіз фішингового повідомлення

Прочитайте повідомлення:

> Ваш акаунт буде заблоковано! Терміново перейдіть за посиланням:
> http://secure-bank-free.xyz

## Потрібно:
- визначити ознаки фішингу;
- пояснити, чому це повідомлення є небезпечним;
- запропонувати правильні дії користувача.

---

# Завдання 3. Безпечний чи небезпечний сайт

Перегляньте характеристики сайту:

- адреса сайту: http://free-bonus-site.net
- багато реклами;
- відсутній HTTPS;
- сайт просить ввести пароль.

## Завдання:
Визначте, чи є сайт безпечним. Поясніть свою відповідь.

---

# Завдання 4. Захист персональних даних

Складіть список правил безпечного використання соціальних мереж.

## У списку повинні бути:
- правила створення паролів;
- правила спілкування;
- захист особистої інформації;
- правила роботи з посиланнями.

---

# Завдання 5. Практична ситуація

Уявіть, що вам на електронну пошту надійшов лист від невідомого відправника з вкладеним файлом.

## Потрібно:
1. Описати можливі загрози.
2. Пояснити, що потрібно зробити перед відкриттям файлу.
3. Запропонувати безпечний алгоритм дій.
