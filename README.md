# Основи інформаційної безпеки (кібербезпека)

---

## Автор
Вікторія Чорна

---

## Опис проєкту

Даний проєкт створений у межах лабораторної роботи з дисципліни:

«Розробка навчальних матеріалів з інформатики з використанням генеративного штучного інтелекту».

Метою проєкту є створення сучасного навчального вебресурсу для вивчення основ інформаційної безпеки та кібербезпеки учнями 10–11 класів.

---

## Мета навчального проєкту

- сформувати базові знання з кібербезпеки;
- ознайомити учнів з основними кіберзагрозами;
- навчити захищати персональні дані;
- розвинути навички безпечної поведінки в Інтернеті.
